# 행운의 숫자 5개

로또 번호처럼 1부터 45 사이의 숫자 5개를 무작위로 추천받을 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 🎲 **숫자 추천** - 버튼 클릭으로 1~45 사이의 숫자 5개를 무작위로 생성
- 🌙 **다크모드/라이트모드** - 사용자의 선호도에 따라 테마 선택 가능
- 📧 **제휴 문의** - 협력 제안을 위한 문의 폼 (Formspree 연동)

## 링크

- 🌐 [웹사이트](https://jeongch112-coder.github.io/product-builder-lecture/)
- 📨 [제휴 문의](https://jeongch112-coder.github.io/product-builder-lecture/contact.html)

## 기술 스택

- HTML5
- CSS3 (CSS Variables 활용)
- Vanilla JavaScript
- Formspree (이메일 서비스)
- GitHub Pages (배포)

## 사용 방법

1. 웹사이트 방문
2. **다시 뽑기** 버튼을 클릭하여 숫자 5개 생성
3. 우상단의 🌙/☀️ 버튼으로 테마 변경
4. **제휴 문의** 버튼으로 협력 제안 전달
